# AI ChatBot Python

## Project Description

This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users and responds to predefined questions.

## Features

- Responds to greetings
- Answers basic questions
- Exit command support
- Console-based interaction

## Technologies Used

- Python

## Run Command

```bash
py chatbot.py
```

## Sample Conversation

You: hello

Bot: Hi! How are you?

You: your name

Bot: I am a Python ChatBot.

You: bye

Bot: Goodbye!
